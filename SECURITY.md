# Security Policy

## Supported Versions

The use of the following versions of this project is currently supported.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of this project seriously. If you find any vulnerability, please report it immediately.

### How to Report

Please do **NOT** report security vulnerabilities through public GitHub issues.

Instead, please report them via email to **vitrsv462@gmail.com** (or your preferred security email).

You should include:
- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact.

### Response Timeline

- **Acknowledgment**: 48 hours
- **Assessment**: 5 business days
- **Fix**: Timeline depends on severity

We appreciate your effort in making this project more secure.
